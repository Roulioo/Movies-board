# Movies-board

Web application about a movies library.

# How to test

1. Go to current folder of project in terminal
2. You need to launch fron-end and back-end
3. First to launch back-end, go to server folder
4. Run "npm install"
5. Run "node index.js"
6. Server will be launched for me is (http://localhost:3000/)
7. Second to launch front-end, go to movies-board folder
8. Run "npm install"
9. Go to public folder
10. Run "npm start"
11. Server will be launched for me is (http://localhost:3001/)
12. Enjoy !
